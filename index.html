HTML (Hypertext Markup Language) is not a programming Language but a markup language. A programming language is a vocabulary and set of grammatical rules for instructing a computer or a computing device to perform specific tasks. Examples of programming languages include:

      Javascript
      Python
      C, C++, C#
      Java
      Kotlin and more.
